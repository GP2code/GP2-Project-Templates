# Quick Start Checklist â€” Data Submission

Use this checklist to ensure your study data is correctly formatted and ready for submission.

---

## 1. Get the Template
- [ ] Download a template found here `templates/`  
- [ ] Review the data dictionary in `templates/`

## 2. Fill in Data
- [ ] One row per participant/observation  
- [ ] Column names unchanged  
- [ ] Categorical values match allowed codes  
- [ ] Missing fields blank or `NA`  
- [ ] No formulas  

## 3. Validate
- [ ] Required variables complete  
- [ ] Column names match exactly  
- [ ] No extra spaces or hidden characters  
- [ ] No merged cells  
- [ ] Saved as `.xlsx` or `.csv`

## 4. Submit via App
- [ ] Access the **QC App** [here](https://gp2-clinical-qc-dot-gp2-data-explorer.uc.r.appspot.com/) 
- [ ] Upload completed `.xlsx` or `.csv` file  
- [ ] Pass all QC checks and submit  

## 5. Data Privacy
- [ ] De-identified data only  
- [ ] Compliant with ethics and data sharing agreements  

## Optional
- Double-check categorical values in the data dictionary  
- Save a copy of the QCed file right before submission
